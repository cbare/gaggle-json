Gaggle-JSON

Metadata

Metadata is a set of key-value pairs used to annotate a gaggle data object. The presence of the metadata key is optional. It can take any form, but conventions should be established for common uses.

        "metadata": {
          "species": "Halobacterium salinarum NRC-1",
          "identifier-type": "VNG",
          "source": "cMonkey"
        }
species Species to which the data pertains, for example 'Halobacterium salinarum NRC-1'.
identifier-type The naming systems from which identifiers are drawn. For example, 'VNG', 'Entrez gene ID', 'Uniprot', etc.
source Source program of object.
citation Citation relevant to data object.
